automatically ban ips

== Description ==

If somebody open site too frequently ban their IPs with .htaccess.

== Installation ==

1. Upload `wp-ban-ip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
